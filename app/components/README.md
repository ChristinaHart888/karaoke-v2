# Components

### About

This folder contains components that may be used between multiple different pages. The list of components are listed below.

### Components

| File Name          | Purpose                                                                               |
| ------------------ | ------------------------------------------------------------------------------------- |
| BootstrapClient.js | Contains the required libraries to use Bootstrap throught the Website                 |
| firestore.js       | Contains the required setup to allow communication with Firebase                      |
| popup.jsx          | A Popup box that can be used on pages that require a popup box to display information |
| Navbar.jsx         | The navbar used for the site                                                          |
